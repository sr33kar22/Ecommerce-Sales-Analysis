📊 E-commerce Sales & Profit Analysis Dashboard

## 📌 Overview
This project analyzes e-commerce sales data to uncover trends, top-performing products, and profitability insights. The dashboard helps identify revenue drivers and highlights products with low profitability.

---

## 🎯 Problem Statement
Businesses often struggle to understand:
- Which products drive the most revenue
- Seasonal sales patterns
- Whether high sales actually translate to profit

This dashboard answers these questions using data-driven analysis.

---

## 📊 Dashboard Preview

![Dashboard](images/dashboard.png)

---

## 🔍 Key Insights

- 📈 Sales show strong seasonality, with peak performance in **November and December**
- 🏆 A small number of products contribute significantly to total revenue
- ⚠️ Some high-selling products generate **low profit**, indicating margin issues
- 🌍 Regional filtering shows variation in performance across markets

---

## 🛠️ Tools Used

- Power BI (Data Visualization)
- DAX (Calculated columns)
- Excel/CSV (Dataset)

---

## 📁 Dataset

- Superstore dataset (sample e-commerce data)
- Includes: Orders, Sales, Profit, Customers, Products, Region

---

## 📌 Features

- KPI Cards (Total Sales, Profit, Orders)
- Monthly Sales Trend Analysis
- Top 10 Products by Sales
- Lowest Profit Products
- Interactive Region Filter (Dropdown)

---

## 🚀 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Use the Region slicer to explore data
4. Analyze trends and product performance

---

## 📎 Project Files

- `dashboard.pbix` → Power BI dashboard
- `dataset.csv` → Source data
- `images/` → Dashboard screenshots

---

## 💡 Conclusion

This analysis highlights that **high sales do not always mean high profit**, emphasizing the need for better pricing and cost optimization strategies.
